# 伊藤勉強用リポジトリ
# Next.jsは使わなさそうなので、途中ですが別リポジトリに移行

完全に個人用ですが、視力検査ウェブアプリを TypeScript + React + Next で作成中です。  
このリポジトリの目的は、モダン技術に関する理解を深めるのと、クリーンアーキテクチャに準拠した実装を構築することです。

## 開発環境

Node v20.9.0  
Cursor Editor v0.18.8

## Getting Started

開発サーバー起動
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
StoryBook
```bash
npm run sb
# or
yarn sb
# or
pnpm sb
# or
bun sb
```

> npmとかyarnとか統一したい...
